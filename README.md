# 一首（ISSHU）

小倉百人一首を暗記するための、依存ライブラリなしの静的Webアプリです。

## 機能

- 全100首の検索・閲覧
- PC：下の句をキーボード入力し、ひらがな一文字ごとに正誤判定
- 全文入力または全句選択が完了した時点で自動判定
- PC：次に入力する一文字だけを表示するヒント
- PC・スマートフォン：回答全文と読みを表示するヒント
- スマートフォン：下の句の前半・後半をそれぞれ4択で回答
- 歌番号（N番〜M番）による出題範囲の指定
- 上の句→下の句、下の句→上の句をランダムに出題
- 正答率の低い歌を優先する復習
- 習得状況と正答率をブラウザに保存
- スマートフォン対応

歌データは[クジラWeb API](https://api.aoikujira.com/index.php?hyakunin=&show=)の百人一首データをアプリ内に同梱しています。
